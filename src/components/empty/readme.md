```html
<empty label="暂无记录" :list="list" />
```


参数名              |  类型         |  默认值       |   说明
--------------------|---------------|---------------|----------
label               | String        |  暂无记录     |   标签
list                | Array/null    |               |   当list为非数组或者数组长度大于0时,显示此组件
